# 🧮 Manipulação de Matrizes em C usando Vetores Unidimensionais

Este projeto foi desenvolvido em linguagem C com base em outro projeto realizado como requisito avaliativo para a disciplina de Estruturas de Dados, tendo o objetivo de manipular matrizes utilizando apenas vetores unidimensionais.

## 📌 Descrição

Via de regra, matrizes em C são representadas através de arrays bidimensionais. No presente projeto, optou-se por uma abordagem diferente, implementando lógica de manuseio de matrizes valendo-se apenas de vetores unidimensionais.

O projeto logrou êxito em exercitar e aprimorar conhecimento em pontos como:

* Organização de memória
* Lógica algorítmica
* Estruturas de dados em baixo nível

## ⚙️ Funcionalidades

* Criação de matrizes
* Acesso a elementos via índice calculado
* Manipulação eficiente de dados em memória

## 🧠 Como funciona

Uma matriz `M` de dimensões `3 x 3` é armazenada em um vetor `v`:

Índice do elemento:

```c
v[i * colunas + j]
```

Onde:

* `i` = linha
* `j` = coluna

## 🚀 Como executar

1. Compile o código:

```bash
gcc main.c -o matriz
```

2. Execute:

```bash
./matriz
```
